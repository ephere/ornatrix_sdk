The purpose of this collection of files is to demonstrate, in a basic way, the usage of the interfaces exposed by the Ornatrix plugin for various hosts
It is not meant to be used for other purposes.

To build the examples on Windows open OrnatrixPluginExamples.sln in Visual Studio 2012 or later.

Set the header and library search paths of 3ds Max SDK and Maya DevKit in LocalBuild.props or in the project properties.
